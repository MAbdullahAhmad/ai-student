# Libraries

Python libraries and its topics / functions covered.

### Python Libraries for AI, ML, DL, CV, and ANNs

| **Library**       | **Primary Use**                                 | **Dependencies**            | **Description**                                                                                                                                  |
|-------------------|-------------------------------------------------|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| **scikit-learn**  | Machine Learning                                | numpy, scipy, joblib        | A popular library for classical machine learning algorithms, including regression, classification, clustering, and dimensionality reduction.      |
| **TensorFlow**    | Deep Learning, ANN, ML                          | numpy, six, protobuf, h5py  | An open-source framework for building and training deep learning models. Developed by Google, it supports various neural network architectures.   |
| **Keras**         | Deep Learning, ANN                              | numpy, scipy, tensorflow    | A high-level neural networks API, written in Python and capable of running on top of TensorFlow, making it easier to build and train deep learning models. |
| **Hugging Face**  | Natural Language Processing (NLP)               | torch, transformers         | A platform providing state-of-the-art NLP models and tools, facilitating easy use and deployment of transformer-based models for text processing. |
| **Diffusers**     | Diffusion Models                                | transformers, torch         | A library for state-of-the-art diffusion models, focusing on image generation and related tasks.                                                  |
| **Transformers**  | Natural Language Processing (NLP)               | numpy, torch, tensorflow, huggingface_hub | A library for working with transformer-based models like BERT, GPT, and T5, supporting a variety of tasks such as text classification and generation. |
| **opencv-python** | Computer Vision (CV)                            | numpy                       | An open-source library providing tools for computer vision tasks such as image processing, video capture, and object detection.                   |
| **numpy**         | Numerical Computations                          |                             | A fundamental package for scientific computing in Python, offering support for large, multi-dimensional arrays and matrices, along with a collection of mathematical functions. |
| **pandas**        | Data Manipulation and Analysis                  | numpy                       | A library providing data structures and data analysis tools for handling and analyzing structured data.                                           |
| **PyTorch**       | Deep Learning, ANN, ML                          | numpy                       | An open-source machine learning library based on the Torch library, used for applications such as computer vision and natural language processing. |
| **Matplotlib**    | Data Visualization                              | numpy, pyparsing, python-dateutil | A plotting library used for creating static, animated, and interactive visualizations in Python.                                                  |
| **Seaborn**       | Statistical Data Visualization                  | numpy, pandas, matplotlib   | A library built on top of Matplotlib that provides a high-level interface for drawing attractive and informative statistical graphics.            |
| **SciPy**         | Scientific Computing                            | numpy                       | A library used for scientific and technical computing, offering modules for optimization, integration, interpolation, eigenvalue problems, and more. |
| **NLTK**          | Natural Language Processing (NLP)               | numpy                       | A comprehensive library for building NLP applications, offering tools for text processing, classification, tokenization, stemming, tagging, parsing, and more. |
| **spaCy**         | Natural Language Processing (NLP)               | numpy                       | An open-source library for advanced NLP, providing pre-trained models and tools for text processing, named entity recognition, and more.          |
| **Statsmodels**   | Statistical Modeling and Testing                | numpy, scipy, pandas, patsy | A library for estimating and testing statistical models, providing classes and functions for the estimation of many different statistical models.  |
| **LightGBM**      | Gradient Boosting Framework                     | numpy, scipy                | A gradient boosting framework that uses tree-based learning algorithms, designed for speed and efficiency, and often used in Kaggle competitions.  |
| **XGBoost**       | Gradient Boosting Framework                     | numpy, scipy                | An optimized gradient boosting library designed to be highly efficient, flexible, and portable, often used in machine learning competitions.       |

### Dependencies and Installation

For installing these libraries, you can use `pip`. Here are some examples:

1. **scikit-learn:**
   ```sh
   pip install scikit-learn
   ```

2. **TensorFlow:**
   ```sh
   pip install tensorflow
   ```

3. **Keras:**
